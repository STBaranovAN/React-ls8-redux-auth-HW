Создайте front-end часть React приложения Chats.
В папке Sever находится backend часть, которая обслуживает выдачу списка чат-комнат, списка сообщений для каждой чат комнаты и принимает с front-end части методом POST объект для создания нового сообщения в выбранной чат-комнате.
Все end-points для обращения AJAX запросами с front-end части прописаны на схеме приложения.
Приложение состоит из трех компонентов, каждый из которых может делать AJAX запросы на backend для получения соответствующий данных.
Для Chatrooms - это список всех ChatRooms
Для Messages - это список сообщений в выбранной комнате
И для PostMsg - это отправка JSON данных для создания нового сообщения в текущей комнате.

Логика работы приложения следующая:
Слева загружается список всех ChatRooms.
На каждую можно нажать и справа в Messages отображаются сообщения выбранного чата.
Если при этом мы напишем новое сообщение в форме внизу, оно отправляется в текущий чат.
